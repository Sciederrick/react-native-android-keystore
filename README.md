# rn-android-keystore

React Native Keystore wrapper for creating cryptographic key pair for signing payloads

## Installation


```sh
npm install rn-android-keystore
```


## Usage


```js
import { multiply } from 'rn-android-keystore';

// ...

const result = multiply(3, 7);
```
